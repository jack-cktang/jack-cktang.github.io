Roon 要點開始⋯⋯玩？

3 February, 2021adminLeave a comment


好多想開始玩CAS嘅師兄都有問我呢個問題，我都答咗好多次，不如喺度講吓點玩啦⋯⋯

Roon 其實係一個音樂檔案管理系統 (Music Library)，有音樂播放功能，支援串流服務，有個好好用嘅使用者介面，分月費及永久付費計劃。https://roonlabs.com/howroonworks 而 Roon 主要分成三個部分，Roon Core 係 Roon 軟件的主要部分，可以安裝在任何硬件上，包括 Windows, Mac, Linux, NAS。Roon Control Apps 就係控制程式，可以裝喺 iOS, Android, Windows, Mac 上。另外可以加埋 Roon Outputs (又叫 End Points)，亦即係 Roon Ready 嘅 Streamer/Speaker/Player 等，合埋便組成一套完整的 Roon 系統。

玩 Roon 第一樣要考慮裝喺邊？Roon Core 可以裝喺任何電腦上，如果冇電腦，可能要買一部 Roon Core 專用電腦，當然可以買支援 Roon 嘅專用 Music Server 例如 Roon Nucleus https://roonlabs.com/nucleus、或者 Innuos Music Server https://innuos.com/en/catalog/go/music-servers-mk3 之類的產品。如果有電腦，可以裝喺自己電腦上面，不過用專用電腦絶對好過普通電腦。另外，如果NAS支援的話，可以裝入NAS上面，不過因爲 Roon 行 Library 嘅時候，需要快速存取資料，所以最好行SSD。如果能力許可的話，最好當然係行專用電腦啦，但如果覺得音響品牌太貴的話，自組電腦也可以的。

如果決定自組電腦的話，首先要清楚這個 Music Library 有多大？如果有大量音樂檔，可能裝硬碟的位置未必足夠，假設這部電腦只可以放一隻硬碟，我建議最好行SSD，依家買 2TB SSD 價錢都可以接受，所以音樂檔少過 2TB的話，其實一隻SSD便足夠。雖然大部分電腦可以行外置硬碟，但小弟就唔建議，因為外置硬碟比較容易出問題。還有，這部電腦要多快？這取決於是否要行 Roon 升頻，如果唔行升頻，i5 已經足夠，如果行升頻，可能要用到 i7。不過小弟覺得可以唔行升頻，用部低耗電CPU和靜音電腦會理想啲。但如果這部電腦冇足夠位放硬碟，可能要考慮加NAS，而 Roon 加NAS係比較方便，有時還方便過內置硬碟。

至於點控制部 Roon，其實好簡單，只要裝個 Roon Remote 落平板電腦就得，如果未有平板電腦的話，可以買部平平地嘅就得，其實手機也可以，但小弟建議要行 iOS，iOS 比較好用。如果已經裝咗 Roon 落 Windows 或 Mac 電腦嘅師兄，同樣地可以用 Windows/Mac 嘅 Roon 去控制聽歌，控制介面是一樣的。

還有最終連接音響系統的地方，首選當然係 Roon Ready 嘅 Streamers，如果只有 USB DAC，也可以用 Roon Core 電腦的USB接DAC。不過還可以用低耗電腦做 End Point (行 Roon Bridge 軟件)，Rasberry Pi 係一個好選擇，https://ropieee.org/ RoPieee 係一個專門行 Rasberry Pi 的OS，簡單易用。至於 Roon Ready 嘅 Streamers 又可以分為有解碼功能或沒有解碼功能，冇解碼功能的 Streamer 要外接DAC，有 SPDIF/AES、USB 和 I2S 的連接方式，當然行 I2S 比較好，不過就唔多選擇，又有兼容問題，而USB連線也可能出現不兼容情況，SPDIF/AES 最穩陣，不過音樂檔最高只兼容24/192，各有優劣。當然也有其他方式出聲，電腦 sound card 也好，HDMI 駁 AV Amp 也好，都係一種方式，不過效果並不是太理想。

實際例子，我和幾位朋友都係咁做：

  1. NAS 裝音樂檔
  2. 獨立組裝電腦行 Roon Core + SSD 硬碟
  3. Streamer + DAC 駁音響系統
  4. LPS 做供電
  5. 光纖網路 + 隔離功能網路交換器 接 Streamer、Roon Core 電腦和NAS
  6. 電處 + 地盒

想玩CAS玩得好聲，唔係一件易事，不過依家有 Roon 可以用，其實比以前方便了很多，亦因爲有 Roon Ready 嘅 Streamers 出現，亦已經簡單很多。有CD機當然好，不過現時的高清或串流未必玩到，而且現時網上音樂平台資源，如 Tidal, Qobuz 的串流服務或下載服務已經盛行，音樂格式亦比CD好，所以係時候要計劃點玩好CAS了。
