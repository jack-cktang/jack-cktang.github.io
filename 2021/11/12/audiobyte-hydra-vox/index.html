【圍物論】領先 1-bit 轉換技術 — AudioByte Hydra-VOX「驚喜」之作

by admin Posted on 12 November, 20216 December, 2023

「驚喜」之作，當然又驚又喜，部解碼新買返嚟一開聲，真係只有驚到流汗，普普通通，聲音一般到唔起眼，有種高科技中伏的感覺。不過到後來當其他器材改善的時候，才發現它的優點，真正的「驚喜」之作。

Audiobyte Hydra-VOX 係一部獨特的 FPGA 解碼器，再加上耳機放大功能，用上同廠的 Hydra-ZAP 超級電容清潔電源供應器，淨睇技術已經想到聲音必然有驚喜。VOX 有專門的 DSP 訊號處理，結合 digital/ analog filters 相位修正技術，再經由 1-bit 處理的分立元件，完全自家技術底下，創造出領先科技的水平，聲稱能夠造到自然聲音的體驗。

開始之時，使用了當時的參考系統接駁，Stenheim 三仔、Soulution 330、Uptone EtherRegen、Intel NUC streamer 等器材，用 USB 接 VOX 開聲，心想… 唔係呀！聲音速度特慢、拖泥帶水，唔緊要，煲多幾日再講。結果⋯ 好少可玩唔夠一個月就將佢放埋一邊，再次攞出嚟玩已經係一年之後嘅事。

驚喜一：就喺前幾個月，見到提及 VOX 有新 Firmware 的討論，於是又試吓將手頭上嘅 VOX 升級，升級之後發現聲音有很大改善，終於有心機再玩好佢。又或者因為現時串流系統已經有了可聞的改善，於是再接上手頭上的器材，有 YG Carmel 2、Vitus SIA-025 mk2、Raspberry Pi 4 streamer、HQPlayer 等，還是用上 USB 接駁，音樂味特別出眾，但總是好像有些缺失似的，咁嘅情況下，又再思考吓其他因素，或者可能同 USB 接駁有關。

驚喜二：既然要從接駁考量，而 VOX 只有三種輸入 USB、SPDIF、I2S，當中應該以 I2S 最理想，不過手頭上沒有 I2S 設備，於是又買了 Singxer SU-6 DDC (其實呢部機想買好耐，只不過未有觸發點) 和 iFi Stream 等器材。當接好 VOX 再開聲嘅時候，感覺當時嘅缺失已經沒有了，換來了自然的聲音，可能因為 I2S 有隔離功能，可以降低 jitter 對系統的影響，從而改善聲音的質素。VOX 有個 DSD 直出功能，當訊號是 DSD256 或 DSD512 輸入時，便可以做到直解，呢個功能相當適合玩 HQPlayer，iFi Stream 可以支援 HQPlayer NAA 行 DSD256，係最理想的軟件升頻直出嘅方法。

驚喜三：玩了一輪 I2S 接駁之後，又因為改變了系統的設置，今次輪到玩 Elac Uni-Fi 書架喇叭，改用 SPDIF 輸入，由 Wattson Emerson Digital 接 Chord M-Scaler 再接 VOX，估唔到原來 VOX 可以支援 384kHz 輸入，今次可以玩盡 M-Scaler 嘅升頻，有興趣可以入去睇吓 【音響拼圖】音響性與音樂性兼備的組合⋯⋯ 爆得起 Elac+NCore 配合 極致升頻 M-Scaler+Hydra-VOX，平價組合可玩到超身價幾倍的聲音，認真有驚喜。其實 VOX 會將 PCM 轉成 delta-sigma 形式去 1-bit 轉換器，而 PCM 轉 DSD 也有出色表現，可能 VOX 嘅 DSP 處理多 bit 運算比較成熟，digital filters 做得相當唔錯。

結論：玩一件器材要玩得好，當然要花上十二分精力和時間，一時三刻未必掌握得到玩器材嘅方法，就因為千變萬化，所以音響才好玩，就算表面好簡單，配搭嘅過程都有好多變數，如果只聽一段短時間就判定器材的好壞，咁就可能過於片面，想要全面的話，就一定要充分了解各種可能性。

*串流參考設定：
串流平台：Tidal, Roon Core (Mac Mini M1) + Roon Bridge + Roon Remote (iPad/MacOS), HQPlayer (Mac Mini M1)
串流系統：Wattson Audio Emerson DIGITAL (SPDIF), iFi ZEN Stream (USB), Singxer SU-6 DDC (I2S)
HydraVOX USB 設定：
Mac Mini M1 (Roon+HQPlayer) -> (Fiber) -> Uptone EtherRegen -> (Ethernet) -> Raspberry Pi (NAA) -> (USB) -> HydraVOX
HydraVOX I2S 設定：
Mac Mini M1 (Roon+HQPlayer) -> (Fiber) -> Uptone EtherRegen -> (Ethernet) -> iFi Stream (NAA) -> (USB) -> Singxer SU-6 -> (I2S) -> HydraVOX
HydraVOX SPDIF 設定：
Mac Mini M1 (Roon) -> (Fiber) -> Uptone EtherRegen -> (Ethernet) -> Emerson Digital (RoonBridge) -> (SPDIF) -> Chord M-Scaler -> (SPDIF) -> HydraVOX
